# Rock Paper Scissors Game 🎮

A simple Rock Paper Scissors game created using Python.

## 📌 About the Project

This is a beginner Python project where the user plays Rock, Paper, Scissors against the computer.

The computer randomly selects one option, and the winner is decided based on the game rules.

## 🚀 Features

- User input from keyboard
- Random computer choice
- Winner determination
- Draw condition handling
- Beginner-friendly Python code

## 🛠️ Technologies Used

- Python
- Random module

## ▶️ How to Run

1. Install Python.
2. Download the project files.
3. Open the terminal or command prompt.
4. Run:

```bash
python rock_paper_scissors.py
